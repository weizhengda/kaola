##网易考拉海购

##技术栈
1. 前端
    - 使用`bootstrap`完成网站前端部分样式搭建
	- 使用`jQuery`完成js交互及ajax网络请求
    - 使用`Less`进行CSS样式预编译
    - 使用`swiper`插件完成自动轮播图
    - 登录、注册页使用伸缩盒子布局
    
2. 后端
	- 使用Express框架
	- 使用ejs模板引擎
	- 使用mongodb数据库
	
3. gulp集成
	- 集成`gulp`完成服务器的自动重启
	- 集成`gulp-htmlmin`对html文件进行压缩
	
	
##前端页面
1. 大首页
2. 列表页
3. 详情页
4. 登录页
5. 注册页
6. 购物车页


##启动服务器

$ node app.js
